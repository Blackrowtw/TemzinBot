TemzinBot
=========

mineflayerを利用した適当bot

Initialize
----------

```
npm install
```

Setup
-----

.env.sample をコピーして .env というファイルを作成し、内容を書き換えてください。

```
MC_HOST="localhost"
MC_PORT="25565"
MC_USERNAME="user@foo.bar"
MC_PASSWORD="password"
```

Usage
-----

```
npm start
```

Update
------

新しいバージョンに対応するには、mineflayerをアップデートしてください。

```
npm update
```
